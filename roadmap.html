Roadmap

Este roadmap es un documento dinámico que ofrece una visión del camino que ha seguido Point y hacia donde se dirige en base a las prioridades de desarrollo y retroalimentación de la Comunidad.

Explorador de bloques

12/04/2018

Puesta en línea de explorador de bloques Iquidus en la URL http://explorer.pointcoin.website:3002

Puesta en línea de sitio web con web wallet

10/04/2018

Se completa desarrollo de sitio web del proyecto y su puesta en línea. El sitio web incorpora una web wallet funcional.

Pool de minería

16/04/2018

Puesta en línea de pool de minería MPOS cortesía de Premia Network en la URL https://pool.premia.network

Desarrollo de App con Web Wallet para Android

16/03/2018

Se publica en Google Play Store la primera App para android con monedero Point integrado como parte del programa de fidelización de clientes Premia Network.

Reinicio de cadena de bloques

15/03/2018

Tras un periodo de pruebas se reinicia la cadena de bloques para comenzar minado y difución del proyecto y crear bloque (bloque 2) de preminado por un monto de 1,000 millones de Point el cual será utilizado para realizar disrtibución entre la población y cubrir costos operativos del proyecto.

Nodos semilla

15/11/2016

Puesta en operación de un nodo semilla y dos nodos de respaldo

Fork de cripto activo y generación de bloque génesis

11/11/2016

Se selecciona una versión previa de Litecoin como código base y manteniendo la prueba de trabajo como algoritmo de consenso.

  • Tiempo entre bloques modificado a 15 segundos
  • Reajuste de dificultad modificado a cada cuatro bloques e implementación de segmentos de código de DigiShield
  • Modificación de algoritmo de recompensa por bloque - A partir del bloque 50,000 se utiliza el incremento consecutivo del bloque procesado (altura de bloque) dividida entre 100,000 para establecer la recompensa.

Por que usar Point

  • Off-Market Se compra/vende fuera de los mercados lo cual elimina la volatilidad y manipulación de precio
  • Velocidad El timepo entre bloques de 15 segundos permite confirmar transacciones de forma ágil en operaciones cotidianas
  • Blockchain propio No depende de cadenas de bloques de terceros
  • Progresive reward La recompensa por bloque progresiva ayudará a mantener estable su valor futuro
